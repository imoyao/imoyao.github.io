      在构建Redis时选择非默认内存分配器是通过设置MALLOC环境变量完成的， 默认情况下 Redis 是使用malloc为libc编译和链接的。
      而libc并不是Linux上默认的分配器，默认的是 jemalloc, 因为 jemalloc 被证明比libc有更少的碎片问题（fragmentation problems）。
